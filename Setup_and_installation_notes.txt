Setup
------------

git clone https://github.com/hannocoetzer/Todo.git
git switch master
cd TodoAPI\
dotnet ef migrations add Eerste
dotnet ef database update

Open C:\temp\todo\TodoAPI\TodoAPI.sln (I used VS 2022)

In TodoAPI\appsettings.json change DefaultConnection



Run in Release mode https (for optimal performance - no debug logging)
	if prompted "click Disable Just my Code and conitnue"


You can check swagger at https://localhost:7144/swagger/index.html
